# Grade-Manager-Console-app-2.0

********** This is 100% C# console app *****
** App uses OOP Principles **
 
 App breakdown:
 
 Classes:
 - School
 - Classroom
 - Student
 - Assignment
 
Features available to user:

Main Menu:
- Add Classroom
- Show Classroom
- Remove Classroom
- Classroom Details
- Exit

Classroom Details Menu:
- Add Student
- Show Students
- Remove Students
- Student Details
- Compare Students
- Show Average Grade
- Best Student
- Worst Student
- Back to main menu

Student Details:
- Add Assignment
- Show Summary
- Remove Assignment
- Show Assignment
- Grade Assignment
- Show Best Grade
- Show Worst Grade
- Back to Classroom Details Menu
